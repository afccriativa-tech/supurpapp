# Kimbuka - Gestão Estratégica Pessoal

**Kimbuka** (Estratégia/Plano em Kimbundu) é uma aplicação web completa para gestão integrada da sua vida profissional e pessoal.

## Funcionalidades

- 📋 **Gestão de Tarefas**: Organize tarefas por categorias (Profissional, AFC Criativa, Ibisner, Pessoal, Estudo, Saúde) com prioridades e subtarefas
- 🎯 **Metas**: Defina metas com progresso percentual e métricas de acompanhamento
- 💰 **Controlo Financeiro**: Registre receitas, despesas e investimentos em Kwanza (Kz)
- 💡 **Banco de Ideias**: Valide ideias com matrizes de impacto e viabilidade
- 📔 **Diário de Reflexão**: Acompanhe seu humor, energia e produtividade diária
- 📚 **Biblioteca de Conhecimento**: Organize livros, cursos e recursos de aprendizagem
- ⏱️ **Temporizador Pomodoro**: Gerencie seu tempo com sessões de 25 minutos
- 📊 **Dashboard**: Visualize estatísticas e progresso em tempo real

## Design

Interface premium em modo escuro com paleta de cores:
- **Carvão**: Fundo principal (#0f1419)
- **Ouro Suave**: Acentos primários (#d4af37)
- **Esmeralda**: Sucesso e positivo (#2d9d78)

## Tecnologia

- HTML5 + CSS3 + JavaScript vanilla
- Armazenamento local (localStorage)
- Responsivo para mobile e desktop
- Sem dependências externas

## Como Usar

1. Abra `index.html` no seu navegador
2. Clique em "Carregar Dados de Exemplo" para ver dados de demonstração
3. Navegue pelas diferentes seções usando a barra lateral
4. Seus dados são salvos automaticamente no navegador

## Estrutura de Dados

- **Tarefas**: Título, categoria, prioridade, data, notas, subtarefas
- **Metas**: Título, categoria, descrição, deadline, progresso, métrica
- **Finanças**: Tipo, origem, categoria, valor, data, descrição
- **Ideias**: Título, descrição, área, status, impacto, viabilidade
- **Diário**: Data, mood, energia, produtividade, reflexão, wins, gratidão
- **Biblioteca**: Título, autor, tipo, status, progresso, tópicos
- **Pomodoro**: Sessões completadas com duração e data
